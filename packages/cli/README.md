# @s11t/cli

The S11t command-line interface owns TOML loading, authoring validation, deterministic compilation, type generation, and artifact emission.

```sh
s11t lint [--config s11t.config.toml] [--format human|json]
s11t build [--config s11t.config.toml] [--check]
s11t inspect namespace:key [--locale ja-JP]
```
